# Hạn Chế Của Bản Demo

- Tính năng tra cứu đơn hàng (Order lookup) chỉ sử dụng file JSON mẫu cục bộ, chưa tích hợp với hệ thống quản lý đơn hàng (OMS) thực tế hay Google Sheets.
- Việc tạo ticket hỗ trợ (Ticket creation) chỉ ghi vào file JSON cục bộ, chưa gửi đến hệ thống helpdesk thực tế (như Zendesk, Jira...).
- Tính năng tra cứu câu hỏi thường gặp (FAQ retrieval) sử dụng dữ liệu mẫu cục bộ.
- Bộ nhớ ngữ cảnh của phiên chat (Session memory) chỉ lưu trữ tạm thời trong tiến trình chạy, phục vụ cho mục đích demo/local.
- UI hiện tại chỉ là giao diện chat demo, chưa phải là bảng điều khiển hỗ trợ (support console) production hoàn chỉnh.
